## 引脚功能：

超声波模块：**PA3**：trig		**PA2**：echo

OLED模块：**PA4**：SCL		**PA5**：SDA

蓝牙串口uart3：**PB10**：TXD		**PB11**：RXD

电机驱动：

**PA8**：PWMA		**PA11**：PWMB

**PB12**：AIN2		**PB13**：AIN1

**PB14**：BIN1		**PB15**：BIN2



电机测速：

**PB6**：A		**PB7**：B

**PA0**：A		**PA1**：B



MPU6050：**PB4**：SCL		**PB3**：SDA		**PB5**：INT
